Сайт фриланс площадка.

1. ДОМАШНЯЯ СТРАНИЦА:
    1.1 Если пользователь не авторизован:
        Выводит 2 кнопки: аторизоваться и зарегистрироваться

        1.1.1 Авторизация. Ввод логина и пароля. Так же есть функция восстановления пароля через почту.
        1.1.2 Регистрация. Сначала выбор типа аккаунта(заказчик либо исполнитель).
            Далее будет представлена соответствующая форма для заполнения. При корректном
            заполнении будет отправлено письмо на почту с ссылкой для подтверждения регистрации.
            После нажатия аккаунт будет считаться активным и после авторизации получит доступ к функциям.

    1.2 Если пользователь авторизован:
        В шапке страницы появятся кнопки: ЛИЧНЫЙ КАБИНЕТ и ВЫЙТИ ИЗ АККАУНТА

        1.2.1 Если авторизован как заказчик:
            На сранице будет представлен список из 5 недавно логинившихся исполнителей и кнопка
            ПОСМОТРЕТЬ ВСЕХ ИСПОЛНИТЕЛЕЙ, которая откроет страницу с перечнем всех исполнителей
            с пагинацией по 10 на одну страницу.

            Нажатие на исполнителя приведет на страницу профиля исполнителя, где будет содержаться информация о нем.

        1.2.2 Если авторизован как исполнитель:
            На сранице будет представлен список из 5 недавно опубликованных задач и кнопка
            ПОСМОТРЕТЬ ВСЕ ЗАКАЗЫ, которая откроет страницу с перечнем всех заказов
            с пагинацией по 10 на одну страницу.

            Нажатие на заказ приведет на страницу заказа, где будет содержаться информация о нем.
            Помимо информации будет кнопка ОТКЛИКНУТЬСЯ, которая отправит запрос на выполнение заказчику.

2. ЛИЧНЫЙ КАБИНЕТ:
    Содержит информацию о пользователе и кнопку РЕДАКТИРОВАТЬ ПРОФИЛЬ, которая откроет форму для редактирования.

    2.1 Если авторизован как заказчик:
        Отображает опубликованные работы(если они есть) и кнопку ОПУБЛИКОВАТЬ ЗАДАЧУ, при нажатии на которую
        будет представлена соответствующая форма
    2.2 Если авторизован как исполнитель:
        Отображает принятые работы(если они есть)

3. ОТКЛИКИ:
    3.1 После того, как исполнитель отправил отклик на выполнение работы, заказчик может посмотреть перечень из
        откликнувшихся исполнителей на странице заказа и выбрать одного из них. В этом случае заказ становится недоступным для всех
        остальных пользователей.
    3.2 На странице заказа и у заказчика и у исполнителя появится кнопка РАБОТА ВЫПОЛНЕНА, если оба участника нажмут на
        эту кнопку, то заказ будет считаться выполненым.
